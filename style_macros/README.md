# egui_component_style_macros

This is a internal macro library designed specifically for `egui_component` to simplify UI style definitions. It is not intended for standalone use.

Please refer to the [egui_component](https://github.com/March-mitsuki/egui_component) documentation for more information.

---

这不是一个可以独立使用的库，它只是一个宏库，它用于`egui_component`内部以简化组件的样式定义。

请查看[egui_component](https://github.com/March-mitsuki/egui_component)的文档以获取更多信息。
